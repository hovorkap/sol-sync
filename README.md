# SkolaOnline ToDo Sync — Home Assistant Add-on Repository

This repository hosts the **SkolaOnline ToDo Sync** Home Assistant add-on.

## Installation

1. In Home Assistant go to **Settings → Add-ons → Add-on Store**
2. Click ⋮ → **Repositories**
3. Add: `https://github.com/hovorkap/sol-sync`
4. Find and install **SkolaOnline ToDo Sync**

## Add-ons

### SkolaOnline ToDo Sync

Scrapes homework assignments from [SkolaOnline.cz](https://www.skolaonline.cz) (Czech school parent portal) and syncs them as reminders to **iCloud Reminders** via CalDAV.

See the [add-on documentation](skolaonline_todo_sync/README.md) for full configuration details.
